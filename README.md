# GGA Website Template

## Installation

### Requirements 
 1. Verify you have NodeJS installed with Node Package Manager
 2. `node --version`
 3. `npm --version`
 
 * `bundle exec middleman server` (to start a server that deploys changes, still must F5 to refresh)`
